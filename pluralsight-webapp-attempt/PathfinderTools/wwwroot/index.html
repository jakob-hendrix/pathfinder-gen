Pathfinder Character Generator

This is going to be a site where you can create and customize a Pathfinder (v1) character.

Intended features include:

  • A user signin
  • The ability to create a new character
  • The ability to manage character levels in a granular level, with retroactive dependency validation
  • Inventory management with the ability to filter by source book
  • State management, including typed-buff and debuffs, buff duration timers, etc.
